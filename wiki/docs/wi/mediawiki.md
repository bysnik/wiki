# MediaWiki

![](https://habrastorage.org/getpro/habr/upload_files/4ff/f4a/6a4/4fff4a6a468afcea825912d1a1e46a54.PNG)

MediaWiki — это бесплатный движок для создания вики-сайтов с открытым исходным кодом, написанный на языке PHP и использующий базы данных (например, MySQL) для хранения контента. Изначально созданный специально для Википедии, он оптимизирован для совместной работы над огромными массивами данных, поддерживая версионность страниц, многоязычность и расширение функционала через тысячи плагинов. Благодаря своей масштабируемости и надежности MediaWiki используется не только в проектах Wikimedia, но и в качестве корпоративных баз знаний в крупнейших организациях мира.
