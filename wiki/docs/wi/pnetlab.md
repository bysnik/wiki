Структура дистрибутива:

```bash
/opt/unetlap/
            /addons
            /data
            /html/"https://github.com/pnetlab/pnetlab_main"
            /labs
            /schema
            /scripts
            /tmp
            /users
            /wrappers
```

Работает на веб сервере Apache2
База данных MySQL
Есть Docker
