# x64dbg

## Назначение

Пакет 'x64dbg' не найден в ALT Linux p11 (x86_64/i586/noarch).

ArchWiki: [страница не найдена] (https://wiki.archlinux.org/title/X64dbg)

## Установка

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install x64dbg
```

### Kali Linux
```bash
sudo apt install x64dbg
```
