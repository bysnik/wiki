# adb

## Назначение

Пакет 'adb' не найден в ALT Linux p11 (x86_64/i586/noarch).

ArchWiki: [страница не найдена] (https://wiki.archlinux.org/title/Adb)

## Установка

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install adb
```

### Kali Linux
```bash
sudo apt install adb
```
