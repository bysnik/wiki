# readelf

## Назначение

Пакет 'readelf' не найден в ALT Linux p11 (x86_64/i586/noarch).

ArchWiki: [страница не найдена] (https://wiki.archlinux.org/title/Readelf)

## Установка

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install readelf
```

### Kali Linux
```bash
sudo apt install readelf
```
