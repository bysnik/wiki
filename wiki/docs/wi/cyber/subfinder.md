# subfinder

## Назначение

Пакет 'subfinder' действительно не найден в ALT Linux p11 (x86_64/i586/noarch).

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install subfinder
```

```bash
sudo apt install subfinder
```
