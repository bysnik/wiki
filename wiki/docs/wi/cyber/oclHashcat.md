# oclHashcat

## Назначение

Пакет 'oclHashcat' действительно не найден в ALT Linux p11 (x86_64/i586/noarch).

## Установка

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install oclHashcat
```

### Kali Linux
```bash
sudo apt install oclHashcat
```
