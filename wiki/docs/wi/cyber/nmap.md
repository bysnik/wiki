# nmap

## Назначение

Пакет 'nmap' не найден в ALT Linux p11 (x86_64/i586/noarch).
https://packages.altlinux.org/ru/sisyphus/srpms/nmap/

## Установка

### ALT Linux
```bash
sudo apt-get update && sudo apt-get install nmap
```

### Kali Linux
```bash
sudo apt install nmap
```
