# Установка драйверов на сканер (SANE)

Были проведены следующие работы:

1. Согласно базе SANE: http://www.sane-project.org/sane-mfgs.html#Z-KYOCERA сканер МФУ находится в статусе `Untested` и использует универсальный бэкэнд `Avision`. Соответственно, ни о какой полноценной поддержке сканера свободными драйверами SANE речи не идёт.

2. Далее был изучен отправленный вам пакет бэкэндов Kyocera, взятый с официального сайта. Данный пакет не содержит специального бэкэнда для сканера. Собственно, сканер - это один из немногих сканеров, которые компания Kyocera для Linux официально не поддерживает.

Изучив доступную информацию и учитывая вышеизложенное, можно сделать вывод, что запустить данный сканер крайне затруднительно. Из свободных источников достоверной информации по успешному запуску данного сканера найдено не было.
